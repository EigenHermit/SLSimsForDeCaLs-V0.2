# SLSimsForDeCaLs-V0.2
This project generates a galaxy-galxy strong lensing system in DECaLS condidtion. It uses a real LRG from DECaLS. The source is sersic simply, and taken from Collect (2015).

required packages:                        

os                
numpy                
scipy                
pandas                
astropy                
open-cv2                 
matplotlib
